# Product Usage Analytics & Funnel Optimization

## About

This project analyzes user behavior, feature adoption, retention, and
funnel performance for a digital product. It uncovers insights such
as: - User engagement patterns (DAU, WAU) - Feature adoption rates -
Retention at Day 1, 7, and 30 - Behavioral correlation with repeat
users - Funnel drop-off points - Hypothesis validation for product
behavior - Recommendations based on data-driven insights

The project combines Python-based analytics with visualizations to
highlight opportunities to increase retention and conversion.

## Key Features

### **User Behavior Analytics**

-   Calculates Daily & Weekly Active Users
-   Tracks engagement trends and peak activity days

### **Feature Adoption Insights**

-   Identifies highly used vs poorly adopted features
-   Helps optimize product onboarding and UX flow

### **Retention Analysis**

-   Computes Day 1, Day 7, and Day 30 retention
-   Detects churn patterns

### **Correlation-Based Behavior Modeling**

-   Identifies which features lead to repeated usage
-   Helps prioritize high-impact features

### **Product Funnel Evaluation**

-   Visual representation of user progression
-   Detects drop-offs in Account → Payment → Transaction flow

### **Hypothesis Testing**

-   Validates assumptions regarding tutorial usage
-   Compares Desktop vs Mobile behavior
-   Analyzes whether feedback correlates with engagement
